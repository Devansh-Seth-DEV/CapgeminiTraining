# Issue Tracking App (Spring Boot Assignment)

#### Case Study:

In a company, employees can raise an issue, in one of the 4 categories:

1. Access permission
2. Cloud issue
3. Desktop issue
4. Other issue

#### Requirements:

- Create entities for employee and Issue. Show mappings

- Write API and Test Cases for:
  
  1. Accessing all issues raised by an employee that are still unresolved.
  
  2. Accessing all issues raised by ANY employee this month.
  
  3. Accessing employee names who raised any issue this month.

> [!NOTE]
> Make sure to use log for logging the requests properly.
